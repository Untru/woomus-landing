- На некоторых коробках — штрих-код или QR-код (стилизованный)
- Цветовая палитра коробок: оттенки бежевого/коричневого (крафт), лента — серо-синяя (фирменный)

**Технические требования к SVG-анимации:**
- Чистый SVG + CSS-анимации (без JS для самой анимации, JS только для запуска/паузы при scroll)
- `<animateTransform>` или CSS `@keyframes` с `transform: translateX()`
- Infinite loop, linear timing
- Optimized: не более 15 КБ в минифицированном виде
- Респонсив: масштабируется через `viewBox`, на мобильных — размещается под текстом
- Smooth 60fps: использовать `will-change: transform` и GPU-ускорение

## 3. SVG-АНИМАЦИЯ "КОНВЕЙЕР" — ПОДРОБНАЯ СПЕЦИФИКАЦИЯ

### 3.1. Общее описание
Зацикленная (infinite loop) SVG-анимация конвейерной ленты с движущимися коробками. Стилистика — flat/semi-isometric, минималистичная, в фирменных цветах.

### 3.2. Элементы сцены

```
┌─────────────────────────────────────────────────────┐
│                    [сканер-луч]                      │
│                        ↓                             │
│  📦  📦   📦    📦  📦   📦    📦                    │
│ ═══○═══○═══○═══○═══○═══○═══○═══○═══                  │
│    ↑ролики конвейера (вращаются)                      │
│ ───────────────────────────────────                   │
│    ↑ станина конвейера                               │
└─────────────────────────────────────────────────────┘
```

### 3.3. Слои (снизу вверх)

1. **Станина конвейера** — два горизонтальных рельса (тёмно-серый `#3A3A3A`), с торцами скруглёнными
2. **Ролики** — серия кругов/цилиндров (`#6B7280`), равномерно расположенных, вращающихся (CSS `rotate`)
3. **Коробки** — прямоугольники с "крышкой" (2-3 типоразмера):
   - Маленькая: 40×35px, цвет `#D4A574`
   - Средняя: 55×45px, цвет `#C4956A`
   - Большая: 70×55px, цвет `#B8845A`
   - На каждой — стилизованные полоски скотча (`#8B6914`) и штрих-код
4. **Лазерный сканер** — вертикальная линия бирюзового цвета (`#00D4AA`) с `opacity` пульсацией, расположена в центре
5. **Цифровые метки** (опционально) — маленькие всплывающие label'ы над коробками при прохождении сканера

### 3.4. Анимация

```css
/* Основное движение коробок */
@keyframes moveBox {
  0%   { transform: translateX(-100px); opacity: 0; }
  5%   { opacity: 1; }
  95%  { opacity: 1; }
  100% { transform: translateX(800px); opacity: 0; }
}

/* Вращение роликов */
@keyframes rollRotate {
  from { transform: rotate(0deg); }
  to   { transform: rotate(360deg); }
}

/* Пульсация сканера */
@keyframes scanPulse {
  0%, 100% { opacity: 0.3; }
  50%      { opacity: 1; }
}
```

- Каждая коробка имеет свой `animation-delay` для равномерного распределения
- Длительность цикла одной коробки: ~6-8 секунд
- Одновременно на экране видно 4-6 коробок
- Ролики вращаются с длительностью 2 секунды

### 3.5. Респонсив

| Viewport | Поведение |
|----------|-----------|
| Desktop (>1024px) | Полная анимация справа от текста hero |
| Tablet (768-1024px) | Уменьшенная, под текстом |
| Mobile (<768px) | Упрощённая (меньше коробок, без меток), под текстом, полная ширина |

### 3.6. Производительность
- Использовать `<use>` для переиспользования элементов коробок
- GPU-ускорение через `will-change: transform`
- `prefers-reduced-motion`: при включённом — статичная картинка конвейера без анимации
